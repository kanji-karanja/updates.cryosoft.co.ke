    COVID19 Kenya Statistics API

    We have set up an API that will allow you to get COVID19 Kenyan updates and allow you to make your own dashboard.

    HTTP Methods

    The API can be accesssed though GET HTTP method

    API Location

    The API can be consumed here at:

    https://covid19.cryosoft.co.ke/api/

    Example Response

    The response of the API is as follows.

    {
"country": "Kenya",
"cases": 2216,
"todayCases": 123,
"deaths": 74,
"todayDeaths": 3,
"recovered": 553,
"active": 1589,
"critical": 7,
"casesPerOneMillion": 41,
"deathsPerOneMillion": 1,
"totalTests": 85058,
"testsPerOneMillion": 1585,
"url": "https://covid19.cryosoft.co.ke/api/",
"documentation": "https://updates.cryosoft.co.ke/documentation/#covid19"
}

    Authentication

    The COVID19 Statistics doesn't require any authentication to get the data.

    String Data Types [Requested]

    You can receive the data that is in form of type Integer as type string if need be. You can simply use the following endpoint by simply adding a query string format=string at the end of the end point as follows:

    https://covid19.cryosoft.co.ke/api/?format=string

    An example response will be as follows:

"country": "Kenya",
